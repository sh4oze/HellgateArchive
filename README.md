# DOOM Tribute Site

Este projeto é um site temático inspirado nos jogos clássicos **DOOM (1993)** e **DOOM II**, desenvolvido por mim **Magno** com foco em estética retrô, identidade visual intensa e homenagem ao impacto cultural da franquia criada pela id Software.

O site foi construído para apresentar, de forma visual e imersiva, elementos marcantes do universo clássico de DOOM, reunindo história, ambientação, arsenal, bestiário, trilha sonora, legado e criadores do jogo em uma experiência única para fãs de FPS antigos.

## Sobre o projeto

A proposta deste site é funcionar como uma página tributo, com estilo sombrio, tecnológico e nostálgico, inspirado na atmosfera dos anos 90 e na linguagem visual dos jogos clássicos de computador.

A navegação foi organizada em seções temáticas, permitindo ao visitante explorar diferentes aspectos do universo de DOOM, desde sua origem até sua permanência como um dos jogos mais influentes da história.

## Principais destaques

- Visual retrô com identidade inspirada em DOOM
- Layout de página única com navegação fluida
- Seção histórica sobre o surgimento da franquia
- Área dedicada aos dois jogos clássicos
- Apresentação do arsenal icônico da série
- Bestiário com alguns dos inimigos mais marcantes
- Seção musical com trilha sonora integrada
- Linha do tempo sobre o legado cultural de DOOM
- Área dedicada aos criadores da id Software
- Animações sutis e efeitos visuais temáticos

## Tecnologias utilizadas

Este projeto foi desenvolvido utilizando tecnologias web fundamentais:

- **HTML5**
- **CSS3**
- **JavaScript**

## Objetivo

O objetivo deste projeto é unir estudo, criatividade e paixão por jogos clássicos, especialmente o universo de DOOM. Além de ser uma homenagem à franquia, o site também serve como exercício de organização estrutural, estilização avançada e interação com JavaScript.

## Estrutura do projeto

O projeto está organizado em arquivos separados para facilitar leitura, manutenção e futuras modificações:

- `index.html` — estrutura do site
- `styles.css` — estilização visual
- `script.js` — interações e comportamento
- `musicas/` — arquivos de áudio da trilha sonora
